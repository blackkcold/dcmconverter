# Security And Privacy / 安全与隐私

## 本地处理 / Local Processing

项目不设计上传接口。浏览器只能读取用户通过文件选择器或目录选择器授权的文件。<br>
The project does not expose an upload path. The browser can only read files explicitly authorized through the file or directory picker.

## 默认匿名 / Default Anonymization

默认导出选项开启 `anonymizeOverlay`。PatientName 显示 `Anonymous`，PatientID 显示 `Hidden`。<br>
The default export option enables `anonymizeOverlay`. PatientName is shown as `Anonymous`, and PatientID is shown as `Hidden`.

## 命名保护 / Naming Protection

导出包名称会清洗非法字符；JPEG 文件名模板只开放主字段白名单，不包含 PatientName / PatientID / PatientSex / PatientAge / BirthDate / AccessionNumber / InstitutionName 等敏感字段。序号前缀和重名兜底后缀由系统生成。<br>
The export package name is sanitized for invalid characters; JPEG filename templates only expose the main-field whitelist and do not include sensitive fields such as PatientName / PatientID / PatientSex / PatientAge / BirthDate / AccessionNumber / InstitutionName. Sequence prefixes and duplicate-name fallback suffixes are generated by the system.

## 风险提示 / Risk Note

V1 不自动识别或清除像素内烧录的患者信息。用户导出前需要确认图像内容。<br>
V1 does not automatically detect or remove burned-in patient information. Users must confirm the image content before exporting.
