Local DICOM JPEG Tool runs entirely in the browser.

本工具完全在浏览器中运行。

Usage:
1. Open the app locally.
2. Select DICOM files or a folder from the file panel.
3. Review metadata and non-diagnostic viewer output.
4. Export JPEG images with anonymized overlay enabled by default.
5. Set the export package name and filename template if you need custom naming.

使用方式：
1. 在本地打开应用。
2. 在文件面板中选择 DICOM 文件或文件夹。
3. 查看 metadata 和非诊断查看结果。
4. 导出 JPEG 图像，默认开启匿名 overlay。
5. 如需自定义命名，可设置导出包名称和文件名模板。

Privacy:
- The app does not upload DICOM files.
- The browser only sees files explicitly selected by the user.
- Exported JPEG images are non-diagnostic renderings.

隐私：
- 应用不会上传 DICOM 文件。
- 浏览器只会看到用户明确选择的文件。
- 导出的 JPEG 图像属于非诊断渲染结果。
